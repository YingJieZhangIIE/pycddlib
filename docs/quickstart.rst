Getting Started
===============

Overview
--------

.. automodule:: pycddlib

Installation
------------

The simplest way to install pycddlib, is to `download <http://github.com/mcmtroffaes/pycddlib/downloads/>`_ a compiled version of the library.

Building From Source
--------------------

To get the source code, clone it with `Git <http://git-scm.com>`_ by running::

    git clone --recursive git://github.com/mcmtroffaes/pycddlib

You can also browse the source code on GitHub: `mcmtroffaes/pycddlib <http://github.com/mcmtroffaes/pycddlib>`_.

Then simply run the :file:`build.sh` script: this will build the library, install it, generate the documentation, and run all the doctests. Note that you need `Cython <http://www.cython.org/>`_ and `Sphinx <http://sphinx.pocoo.org/>`_.
